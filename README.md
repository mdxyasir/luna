<h2>How to use</h2>

### Setup
 - Run ```install.bat```
 - Open ```config.json``` and enter your bot's token
 - (Optional) You can also change the bot's prefix in the ```config.json``` file
### Running the bot
 - Run ```start.bat``` and your bot should be online!
